# Digital Health Nursing Toolkit

A collection of resources, guides, tools, and educational materials focused on:

* Digital Health
* Nursing Informatics
* Healthcare AI
* Telemedicine
* Electronic Health Records (EHR)
* Clinical Documentation
* Healthcare Innovation

## About

This repository is maintained by Mirsad Haliloviq.

The goal of this project is to help healthcare professionals, nursing students, and digital health enthusiasts learn about healthcare technology and digital transformation in healthcare.
